# Discord Lua Obfuscator 🛡️

Obfuscate Lua code with advanced techniques.

## Usage

1. Attach .lua file
2. Type `!obfuscate`
3. Download obfuscated code

## Features

- Variable renaming
- Number encoding
- String splitting
- Dead code insertion
- Function wrapping
- Anti-debug protection

## Commands

- `!obfuscate` - Obfuscate attached file
- `!help` - Show help

Max file size: 1MB
